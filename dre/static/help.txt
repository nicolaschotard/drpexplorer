<h3>DRE</h3>
<ul>
<li>The Data Release Explorer is a a stand-alone interactive way to (you guessed) help explore and grasp the wealth of information inside SNfactory's Data Releases</li>
<li>It aims to make you more productive in your science analysis using the [proto|I|E]DRs</li>
<li>But will not write your poster/letter/paper/conference proceeding</li>
</ul>
   
<h3>DB</h3>
<ul>
<li>DRE uses an internal sqlite DB, which is generated from the META.pkl the first time it is run</li>
<li>You can merge your own analysis results into the DB from a SnfMetaData compatible file</li>
<pre>
./runDRE.py --merge http://snovae.in2p3.fr/pereira/rui.yaml
</pre>
<li>You can tinker directly with Django DB queries using <pre style='display:inline;'>--shell</pre></li>
<pre>
./runDRE.py --shell
  
In [1]: from DB.models import Target
  
In [2]: Target.objects.get(name='SN2005el').spectra.count()
Out[2]: 21
</pre>
<li>A Django DB browser is <a target='_blanck' href='/browse/'>available</a></li>
</ul>

<h3>Hubble diagram</h3>
<ul>
<li><pre style='display:inline;'>hubblizer</pre> outputs can be merged directly into the DB, activating the 'Hubble diagram' tab</li>
<pre>
hubblizer $DRPATH/META.pkl -s training,validation -o myHD.pkl --strip
./runDRE.py --merge myHD.pkl
</pre>
</ul>

<h3>Tips & tricks</h3>
<ul>
<li>Zoo-like view for the nostalgics is under <a target='_blanck' href="/zoo"><pre style="display:inline;">/zoo/</pre></a> and you can also use <pre style="display:inline;">/lc/sn_name/</pre> (<a href="/lc/SN2005el/">like this</a>)</li>
<li>Most of the plots
accept toggling of data series by clicking in the corresponding legend "square"</li>
<li>Double-clicking in an empty area of the main toolbar <i>should</i> open the DRE fullscreen</li>
<li>Tabs can be bookmarked and accessed directly wich anchors like <a href="/DRE/#spectra">#spectra</a> (may need a reload)</li>
<li>Everything should work properly under Mac, Linux and Windows in a sane browser</li>
</ul>
